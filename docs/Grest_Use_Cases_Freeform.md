# RACEN Stress-Test: GREST Policy & FAQ Information-Seeking Use Cases

These scenarios are crafted for retrieval QA stress-testing against the following GREST pages:

- FAQs: https://grest.in/pages/faqs
- Warranty: https://grest.in/pages/warranty
- Returns/Refund/Cancellation: https://grest.in/pages/returns-refund-cancellation
- Shipping: https://grest.in/pages/shipping
- Terms & Conditions: https://grest.in/pages/terms-and-condition

**Design:** 50 use-cases × ≥5 natural-language query variations each. Each is tied to a source page and framed as a user intent you can map to answer snippets, citations, and policy logic.
